# my-startup-nest

[NestJS](https://docs.nestjs.com/)の練習用リポジトリ

## コマンドリスト

- コントローラーの作成

```
$  yarn nest generate co cats
```

- Providerの作成

```
$ yarn nest generate s cats
```

- Interfaceの作成

```
$ yarn nest generate interface cats/interfaces/cat
```

- Moduleの作成

```
$ yarn nest g module cats
```

- Middlewareの作成

```
$ yarn nest g mi common/logger
```
